PROYECTO: VEL-AI Agenda — Spa Bella

Descripción:
Este proyecto corresponde al primer bot funcional de VEL-AI Agenda para un negocio de citas.
El sistema permite atender clientes por WhatsApp, responder servicios y precios, agendar citas,
reagendar, cancelar y permitir consultas privadas del dueño.

Producto:
VEL-AI Agenda

Cliente / piloto:
Spa Bella

Tecnologías utilizadas:
- Python / Flask
- Twilio WhatsApp
- Claude / Anthropic
- Supabase
- Railway

Archivos principales:
- arrancar.py: archivo principal del bot
- requirements.txt: dependencias del proyecto
- Procfile: configuración para Railway
- .env: variables sensibles del entorno, no compartir
- backups: versiones anteriores del código

Estado actual:
Bot funcional en prueba / piloto.

Notas importantes:
No compartir archivos .env ni claves API.
No enviar esta carpeta completa a clientes.
Mantener una copia de respaldo antes de modificar el código.
